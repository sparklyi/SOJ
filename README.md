# SOJ - 看似没有特点，实则确实没有特点的开源OJ系统:)
## 项目简介
- SOJ是一款OJ系统!
- 采取wire依赖注入解耦,采取经典三层架构(handle+service+repository)

## 技术栈
- Golang
- gin web框架
- gorm ORM框架
- wire 依赖注入
- RabbitMQ 消息队列
- MySQL 关系数据库
- MongoDB 文档数据库
- redis 缓存数据库
- judge0 测评+沙箱
- 腾讯云COS 对象存储
- Docker 容器化
- ...(项目还没写完)
